# MoE 전체 herb 추천 (14종)

`predict_moe.py`로 14개 herb 각각의 top-12를 뽑았습니다. 모델 **ThreeExpertMoE**.

- `score`: 3개 LLM 추천 확률 평균 · `gate`: 축별 반영 비율(합=1) · `ax`: 활성 축 수

- ⚠️ MoE는 참고용. 실제 추천은 B5(`predict.py`). 게이트가 대부분 조리로 붕괴하는 걸 확인용.

## 전체에서 보이는 패턴

**게이트가 herb 무관하게 조리(cook)로 쏠립니다.** 14개 herb 모두 3축 활성 조합의 평균 게이트가
cook 0.62~0.71 / flavor 0.16~0.25 / prop 0.10~0.19입니다 — herb마다 거의 같습니다. "재료별
전문가 혼합"이 실제로는 일어나지 않는다는 증거입니다.

**전통 축이 살아나는 곳은 약선 조합뿐입니다.** gate_prop이 0.9 이상인 사례를 모으면 전부 한방
배합입니다:

| herb + 재료 | gate(cook/flav/prop) | 해석 |
|---|---|---|
| 당귀 + 용안육 | 0.00/0.00/1.00 | 쌍화탕 계열 |
| 당귀 + 인삼 | 0.00/0.00/1.00 | 보약 배합 |
| 당귀 + 대추 | 0.00/0.09/0.91 | 약선 기본 |
| 생강 + 대파 | 0.00/0.01/0.99 | (조리·향미 부재) |
| 대추 + 용안육 | 0.00/0.00/1.00 | 약선 |

이들은 RecipeDB·FlavorGraph에 없어 `axes<3`이고, 남은 전통 축이 100%가 됩니다. **한방 약선
조합에서만 전통 축이 주력이 되는 것**이 MoE 게이트로 확인됩니다.

⚠️ **데이터 품질**: `tang orange crystal`(시나몬 12위) 같은 미정규화 영문이 일부 남아 있습니다.
앞서 보고한 미해결 영문 8건 중 하나로, 최종 정제 시 제거 대상입니다.
